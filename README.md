# libgit2

A Dart workspace providing idiomatic Dart bindings to
[libgit2](https://libgit2.org), the portable C implementation of Git.

| Package                                 | Description                                       |
|-----------------------------------------|---------------------------------------------------|
| [`libgit2_dart`](packages/libgit2_dart) | Idiomatic Dart bindings to the libgit2 C library. |
